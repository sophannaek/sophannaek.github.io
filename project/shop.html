Web Application: iShop online Shopping Web Application

Personal Project

June 2019 - Present | Long Beach, CA

image of the final product here !

OVERVIEW

iShop is a web application for an online store where users can shop and sell thousand products without have to leave their home offices.

INTRODUCTION

iShop is an improvement/continuation of my web development class final project. For the final presentation, I created a website illustrated the CRUD. After the class ended, I have extended this simple website to a mini version of AMAZON where customers can find various products. This project was inspired by the gigatic ecommerce: AMAZON.

Homepage

It is the shop main page where all products will be listed. It is accessible by all users with/witout login.

Login

Without login, user can still view all listed products in the main homepage. By login as a customer, user can view all the listed products and purchase them, they can also view, edit their carts. After a sucessful login, the seller can list their new products or edit/delete their existing products.

SignUp

There are two types of user that can be signed for: seller or customer account

User Profile

This feature is available to both type of users. They can edit their profile information.

Add/Edit Product Page:

This feature is only accessible to the seller. This user can add, make change, and delete the product.

Shopping Cart
