GUI Application: Search Engine

Class Project

September 2019 - Present | Long Beach, CA

OVERVIEW

Search Engine is a Java GUI application where users can search for the phrase query they've entered. The user can select the corpus they would like to search for their queries.

INTRODUCTION

This search engine is an on-going project. It currently reads all the content of each document in the selected corpus and index those words into the index. The index is currently processed on RAM only, but it will be implemented into disk in the future, so the index building process should occurs only once.

IMPLEMENTATION

Tools: Eclipse IDE, org.simple Library, Command line
Language: Java
