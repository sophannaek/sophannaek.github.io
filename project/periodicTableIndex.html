A Web Application: Periodic Table

Personal Project

Summer 2019

PROJECT DETAILS

This personal project was inspired by how much I enjoyed learning Chemistry when I was a undergraduate. It is one of my most favorite subjects. When I used to be a tutor, I've always wanted to create something that my students can have fun and learn chemistry. This project consists of three sub applications, a molar mass calculator, periodic table info, and a quiz. In this project, all data is acquired by using the open source periodic table API. For a complete source code, please see here.

Technology Used

  • AJAX, HTML5/CSS, JavaScript
  • Periodic Table API
