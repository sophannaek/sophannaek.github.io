PROJECT DETAILS

iFinder is a platform where users can find places for their favorite food or look for outdoor activities at their target location. For a complete source code, please see here.

TECHNOLOGY USED

  • JavaScript, ReactJS
  • Yelp API, OpenWeatherMap API

FEATURES

  • Search for places/activities at desired location.
  • See current weather at target location.
