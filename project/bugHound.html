A Web Application: BugHound Software Tracking Report

Software Test & Verfication Class Project

Spring 2020 | CSU, Long Beach

PROJECT DETAILS

Bughound is a web application that tracks software bug reports. Employees with the granted access can add, update, delete and search for bug reports. For a complete source code, please see here.

Technology Used

  • HTML5/CSS, JavaScript, PHP, SQL
  • MAMP, XAMMP

Feature

  • User credential authentication
  • Add/edit/delete/search a software bug for all users
  • Access to the database maintenance is limited for admins only [users with user leve of 3 or higher ]
  • Export in XML and ASCII text for each table in the Bughound database
  • Input Validation
